# Arcane Equipment

A Vintage Story mod inspired by named magic items from DnD and Pathfinder. Each item is uniquely craftable and comes with its own passive or toggleable magical effect that interacts visibly with the world.

## Vision

Rather than a generic enchantment system, this mod adds specific named magic items — each with a hand-crafted effect that other players can see and react to. The goal is to enrich roleplay on multiplayer servers with tangible, world-affecting magic.

## Current Status

Work in progress. Not yet ready for production servers.

**First item in development:** Frost Walker Boots — passively freeze nearby water into temporary ice while equipped.

## Planned Features

- Wearable equipment with passive always-on effects
- Toggleable active effects
- Craftable via in-game recipes
- Multiplayer and server compatible
- Future: scrolls, spellbooks, mana system (out of current scope)
